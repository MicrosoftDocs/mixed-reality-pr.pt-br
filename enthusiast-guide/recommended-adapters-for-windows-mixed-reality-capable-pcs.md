# <a name="recommended-adapters"></a>Adaptadores recomendados
> [!NOTE]
> Atualizaremos, fazendo adições e podem ser revisando essas recomendações de adaptador de PC do Windows Mixed Reality. Consulte novamente regularmente para obter as últimas novidades.

Seu [PC compatível com realidade mista do Windows](windows-mixed-reality-minimum-pc-hardware-compatibility-guidelines.md) pode não ter o HDMI completo e as portas USB 3,0 necessárias para conectar o headset de imersão. Se esse for o caso, você precisará de adaptadores para conectar seu headset. Alguns adaptadores também podem exigir a adição de um acoplador para se conectar ao seu PC e/ou ao cabo do headset. Abaixo você encontrará uma lista de tipos de adaptadores que talvez precise e algumas recomendações sobre modelos de adaptadores específicos.

## <a name="windows-mixed-reality-compatible-usb-bluetooth-adapters"></a>Adaptadores USB Bluetooth compatíveis com o Windows Mixed Reality

Se o headset foi lançado após 2018, os adaptadores não são necessários e os controladores emparelham com o headset.

Os dispositivos fornecidos no 2017 ou anterior precisam de um adaptador para os controladores. Se o seu computador não incluir o Bluetooth para emparelhar [controladores de movimento do Windows Mixed Reality](controllers-in-wmr.md#Pair-motion-controllers), você poderá usar um adaptador como o adaptador plug-in USB Bluetooth 4,0 de baixa energia micro.

> [!NOTE]
> A Microsoft não fabrica, testa, certifica nem garante a funcionalidade, a confiabilidade ou a qualidade de qualquer adaptador. Para dúvidas, preocupações ou problemas com qualquer adaptador, você deve entrar em contato com o vendedor ou com o fabricante do adaptador.

## <a name="using-external-usb-30-hubs-with-windows-mixed-reality-headsets"></a>Usando hubs USB 3,0 externos com headsets de realidade mista do Windows

Os headsets de realidade mista do Windows podem ser usados com um hub USB 3,0 externo se o seu PC não tiver portas USB abertas suficientes. O hub USB 3,0 externo **deve ser ligado externamente por um adaptador CA** para fornecer energia suficiente para o headset.

Além disso, observe as seguintes diretrizes para o uso de um hub USB 3,0 externo: 
* O hub USB deve ser um hub USB 3,0.
* O hub USB 3,0 deve estar conectado a uma porta USB 3,0 em seu computador.  
* O hub USB 3,0 deve ser conectado externamente (ter um adaptador AC que forneça energia extra).
* O Hub SuperSpeed de 10 portas de USB 3,0 com o adaptador de energia 48W é conhecido por funcionar bem com headsets de realidade mista do Windows e a maioria dos PCs.
